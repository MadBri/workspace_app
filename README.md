# WorkSpace App
Aplicación full-stack React + NestJS con autenticación JWT y drag & drop.

## Stack
- Frontend: React, Vite, TailwindCSS, dnd-kit
- Backend: NestJS, TypeORM, SQLite, JWT

## Correr el proyecto
### Backend
cd backend && npm install && npm run start:dev

### Frontend  
cd frontend && npm install && npm run dev
